Project Overview: Configure, Price, Quote Software Solution

Database Designer & Software Developer:Nadia Monsefi

I developed a comprehensive CPQ and order management system for Iran ​Tablo under a personal contract. Iran Tablo, a leader in engineering, manufacturing, and installation of power distribution and control systems with over five decades of experience, is the largest and most trusted manufacturer of electrical panels in Iran. The company previously relied on Excel files for generating price proposals, which were prone to errors. My custom-built system replaced these Excel sheets, providing a robust solution designed to streamline complex proposal creation, cost estimation, project management, and change tracking.

Problem Statement:

The company’s existing process relied heavily on large Excel files for proposal creation, resulting in:

  • Frequent errors in cost estimation.
  • Slow performance and delays in responding to client requests.
  • Difficulty in managing frequent changes to project specifications.

Solution:

I designed and developed a custom software solution using .NET technologies and Single Page Application using Angular to address the following key needs:

  • Cost estimation and project management.
  • Change tracking for evolving project specifications.
  • Faster, more accurate proposal generation, tailored to specific client requirements.

Key Features:

  • Database Design: Structured the database to efficiently manage product data, cost variables, and project details.
  • Order Management System: Automated workflows for proposal generation, approval processes, and tracking project changes.
  • Customizable: Built-in flexibility to accommodate frequent changes in client requirements.

Technology Stack:

  • Backend: .NET Framework, C#
  • Database: SQL Server
  • Frontend: Angular, Bootstrap
  • Version Control: Git
  • Project Management: Agile with iterative development and continuous feedback loops from the client.
  • Reports: Stimulsoft Report

Challenges Addressed:

  • Reduced error rates and processing times by moving away from Excel.
  • Allowed for real-time updates and more efficient client communications.

Development Process:

  • Initial Phase: Requirement gathering and database design.
  • Development Phase: Iterative development and testing cycles.
